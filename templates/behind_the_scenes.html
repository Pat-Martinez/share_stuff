Behind the Scenes:

Background, Techie stuff, Credits, etc.


BACKGROUND

This web app was my student project for the PDX Code Guild. The intention was to learn how to build a dynamic, database dependent site using the Model-View-Controller approach. Some other goals were to learn how to use forms, how to use and modify a responsive framework like Bootstrap, how to incorporate Javascript/jQuery elements, and how to host a live site.

This is a work in progress so I'll continue to improve this site and add features.



TECHIE STUFF

Backend

  • Language & Framework: Python 2.7 & Django 1.6. Six models: Member, Moderator, Item, Group, BorrowRequest, JoinRequest
  • Database: PostgreSQL
  • Host & Server: Webfaction running Apache

Frontend

  • Bootstrap 3
  • Javascript & JQuery features
    • Two scripts to make it harder for webcrawlers to harvest email.
    • Various alert boxes, such as confirm deletion of an item.
    • Google Analytics
  • Photoshop Elements

To view the code, visit my GitHub account.

BUGS

  • When a member joins more than one group all sorts of craziness can occur on the member page (home page).
  • File validation of image files (item images, group images, profile images) hasn't been included, thus non-image files can be uploaded. Also need to set a file size limit.
  • Zip code validation hasn't been included on the registration page, thus invalid zipcodes can be added.
