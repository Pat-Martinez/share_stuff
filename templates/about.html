About Share Stuff

My first Web App created with Django.

This web app was my student project for the PDX Code Guild. I had so much fun, and moments of maddening frustration, creating this project in Python and Django. I couldn't have done it without the generous help of Chris Freeman, Chris Jones, and Ben Kulp. Thanks guys!

This is a work in progresss so I'll continue to improve this app and add features.


To view the code, visit my GitHub account.

BUGS
